# src/

Bu dizin gelecekteki backend modülleri ve iş mantığı için ayrılmıştır.

## İçerik

- `escrow/` - Escrow entegrasyonu (opsiyonel)
- `index.js` - Eski backend kodu (artık kullanılmıyor)
- `main.js` - Eski backend kodu (artık kullanılmıyor)
- `supabaseclient.js` - Supabase istemcisi (opsiyonel)

## Not

Ana Express server artık kök dizindeki `server.js` dosyasındadır.
Bu dizindeki dosyalar gelecekte backend geliştirmesi için kullanılabilir.
